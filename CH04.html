第四單元：感官、肢體與健康障礙學生的教學實務

本週課程將擴展師資生的視野，探討聽覺、視覺、肢體及健康障礙學生的特殊教育。我們將以神經多樣性為基礎，運用通用學習設計(UDL)與認知負荷理論，深入分析學生的學習需求，並探討專業化的支持策略、輔助科技的應用與課程調整的原則。

教學的核心概念框架

神經多樣性

視角轉換：從「這個學生有什麼問題？」轉向「他的大腦如何運作？」。承認並尊重所有大腦的自然變異，將障礙視為一種差異，而非缺陷。

通用學習設計(UDL)

教學設計的操作手冊。提供多元的表徵、行動與表達、參與方式，從一開始就為所有學習者設計，而非事後補救。

認知負荷理論

心智的瓶頸管理。教學的目標是降低無關的「外在負荷」，管理必要的「內在負荷」，以釋放學生的心智資源，投入於建構知識的「增益負荷」。

互動式神經認知洞察

點擊下方按鈕，探索不同障礙特質背後的大腦可塑性與認知意涵。

聽覺障礙：跨感官重組的大腦

  • 神經科學洞見：在先天聽障者的大腦中，原用於處理聲音的「聽覺皮質」並不會閒置，而是會被視覺系統「徵用」，以增強周邊視覺的注意力。這稱為「跨感官可塑性」。
  • 認知意涵：由於缺乏聽覺的「偶然學習」（如無意間聽到對話），學生對於抽象概念的「基模」（心智模型）可能是片斷或不完整的。
  • 教學啟示：教學必須極度「具體化」與「視覺化」，透過多感官經驗幫助學生建立穩固的概念基模，這正是UDL「多元表徵」原則的體現。

視覺障礙：重塑的感官地圖

  • 神經科學洞見：盲人在閱讀點字時，其大腦的「視覺皮質」會被活化來處理觸覺訊息。大腦的功能地圖被重新繪製，以適應新的感官輸入。
  • 認知意涵：可能出現「語意不合」現象（能背誦定義但無真實概念），因為其知識基模建立在語言而非具體感知經驗上。其空間概念的建立更依賴序列性處理，而非視覺的平行處理，認知負荷更高。
  • 教學啟示：教學需強調「做中學」與「直接經驗」。擴充核心課程（ECC）之所以重要，是因為它提供了建構真實世界基模的必要工具。

肢體障礙：身體、心靈與可塑的大腦

  • 神經科學洞見：即使中樞神經受損，大腦的「運動皮質」仍具可塑性。透過持續、有目標的復健練習，大腦可以重組功能，由健康的腦區接管受損區域的功能。
  • 認知心理學意涵：學生的核心挑戰常在於因身體限制而產生的負面「身體意象」與自我概念，這會直接影響學習動機與社會參與的意願。
  • 教學啟示：支持的核心是「無障礙」，不僅是物理環境的無障礙，更是心理與社會參與的無障礙。彈性的教學與評量方式能降低外在認知負荷，讓學生能展現其真正的能力。

聽覺障礙學生

單元十：聽覺障礙學生的學習與輔具應用

定義與鑑定

學理定義： 介紹聽力損失的鑑定標準，並說明新鑑定基準納入「單側聽損」的意義。此外，特別關注常被忽略的「微聽損」。

法規依據：《特殊教育學生及幼兒鑑定辦法》第五條

本法第三條第三款所稱聽覺障礙，指由於聽力損失，致使聽覺功能或以聽覺參與活動之能力受到限制，影響參與學習活動。

前項所定聽覺障礙，其鑑定基準依下列各款規定之一：

  • 純音聽力檢查結果，聽力損失達下列各目規定之一：
    • 優耳五百赫、一千赫、二千赫、四千赫聽閾平均值，未滿七歲達二十一分貝以上；七歲以上達二十五分貝以上。
    • 任一耳五百赫、一千赫、二千赫、四千赫聽閾平均值達五十分貝以上。
  • 聽力無法以前款純音聽力測定時，以聽覺電生理檢查方式測定後認定。

特別關注：微聽損 (Minimal Hearing Loss)

「微聽損」是指輕微、單側或高頻的聽力損失。這些孩子常被誤認為「注意力不集中」或「反應慢」，因為他們在安靜環境下似乎聽得不錯，但在噪音中或遠距離聽講時就會出現困難，導致學習和社交上的隱性障礙。及早發現與介入至關重要。

成因分類

依受損部位區分

  • 傳導性聽損：外耳或中耳的聲音傳導路徑發生問題，如耳垢阻塞、中耳炎。通常可透過醫療或手術改善，聲音聽起來會變小聲，但較不失真。
  • 感音神經性聽損：內耳（耳蝸）或聽神經受損，導致聲音的清晰度與響度都受影響。這是最常見的聽損類型，通常是永久性的，需配戴助聽器或人工耳蝸。

依發生時間區分

  • 先天性聽損：出生時即有的聽力損失，可能由遺傳、懷孕期間病毒感染等因素造成。
  • 後天性聽損：出生後因疾病（如腦膜炎）、噪音、藥物、外傷等因素造成的聽力損失。

聽覺系統的運作與評估

第一站：聲音的接收與傳導

  • 外耳 (Outer Ear)：包含耳廓和耳道，負責收集聲波並將其導入耳內。
  • 中耳 (Middle Ear)：包含耳膜和三塊聽小骨。聲波震動耳膜，帶動聽小骨將振動放大，傳遞至內耳。
  • 內耳 (Inner Ear)：核心是耳蝸，一個充滿液體的螺旋狀結構。聽小骨的振動在耳蝸內產生液體波動，刺激數千個微小的毛細胞，將機械能轉換為電化學訊號。

第二站：大腦的解讀之旅 (中樞聽覺路徑)

從內耳毛細胞發出的神經訊號，會經過多條複雜的路徑上傳至大腦。主要路徑分為三條：

a. 路徑一 (主要到對側)：
毛細胞 → 脊髓神經節 → 同側腹側/背側耳蝸核 → 經背側聽紋交叉至對側 → 外側丘系 → 下丘 → 內膝狀體 → 對側大腦聽覺皮質。

b. 路徑二 (到同側)：
毛細胞 → 脊髓神經節 → 同側腹側耳蝸核 → 同側上橄欖核 → 同側外側丘系 → 下丘 → 內膝狀體 → 同側大腦聽覺皮質。

c. 路徑三 (交叉路徑)：
毛細胞 → 脊髓神經節 → 同側腹側耳蝸核 → 經斜方體交叉至對側上橄欖核 → 對側外側丘系 → 下丘 → 內膝狀體 → 對側大腦聽覺皮質。

肌筋膜理論將人體視為一個由結締組織貫穿全身的整體網絡，而非單純肌肉與骨骼的組合。筋膜(Fascia)這層薄膜包覆著肌肉、神經、內臟，形成了多條「肌筋膜經線」，負責傳遞張力與力量。對於身心障礙者，特別是肢體障礙學生而言，理解此系統至關重要，因為局部的肌肉張力問題或活動受限，常會沿著這些筋膜路線，引發全身性的姿勢歪斜與代償問題。

課程調整的理念與實踐

課程調整的發展週期

課程調整皆遵循一個嚴謹且環環相扣的發展週期，此週期確保了所有教育介入措施皆基於實證且具備高度的個別化。此流程可細分為以下四個階段：

全面性的學生評估

每個個案的起點均為一份詳盡的「個案說明」，內容涵蓋學生的認知功能、健康狀況、感官功能、學業表現及心理社會特質等多面向的資料 。此評估不僅僅是分數的羅列，而是對學生作為一個完整個體的質性描述，為後續的分析奠定基礎。

優弱勢分析

此階段將評估資料轉化為對學生學習特質的深入洞察。手冊強調平衡觀點，並陳學生的優勢能力與面臨的挑戰，以形成一個整全的圖像。例如，國小學習障礙個案「小芸」的分析中，其優勢（如良好的記憶背誦能力、母親的全心支持）與弱勢（如文本理解困難、缺乏自信）被並列呈現，這使得介入策略能揚長補短，而非僅僅聚焦於缺陷 。

需求評估

此階段是將優弱勢分析轉化為具體教育需求的關鍵橋樑。學生的弱勢被重新定義為待滿足的「學習需求」。例如，小芸的文本理解困難，對應到「提升語文能力」的需求；國中自閉症個案「葉生」的人際互動困難，則對應到「提升情緒辨識及人際互動的能力」之需求 。此轉換確保了教育目標的針對性與功能性。

目標設定與介入規劃

最終，評估出的需求將直接連結至具體、可測量的IEP學年與學期目標，並據此規劃相應的課程調整策略。此流程展現了一種診斷-處方的醫學模式精神。IEP的品質與成效，深刻地取決於初始評估的深度與準確性。教師在此模式中的角色，不僅是教學執行者，更是專業的教育診斷者。每一個調整決策，例如為無口語能力的重度障礙個案「阿祥」選擇「替代性」溝通系統，並非隨機選擇，而是基於其「無口語」此一診斷結果所開立的精準教育處方。這意味著，有效的特殊教育實踐，其核心在於教師基於證據進行專業判斷的能力。

課程調整的核心原則與三大面向

手冊提出的所有課程調整策略，皆可歸納為四個核心原則。這些原則如同一個工具箱，教育團隊可根據學生的個別需求與學習情境，靈活地選擇與組合運用。

簡化 (Simplification)

降低學習內容的認知複雜度，使其更易於理解與吸收。例如，對於葉生（輕微缺損），簡化可能意味著先從具體詞彙教學入手，再進入抽象譬喻的理解。而對於阿祥（嚴重缺損），簡化則是將總體識字量目標從1800字大幅下修至300個高頻常用字。

減量／分解 (Reduction/Decomposition)

減少學習的份量，或將一項複雜的技能拆解成數個更小、更易於管理的步驟。例如，小芸的數學課程將「解決兩步驟應用問題」此一目標，先分解為「解決第一步驟應用問題」，待其精熟後再進入下一階段。在「夢幻全壘打」的教學示例中，寫作任務所需的語詞數量也被明確地減少，以降低學生的寫作負擔。

替代 (Substitution)

採用不同的方法、教材或輔助科技，來達成實質等同的學習目標或評量方式。此原則在阿祥的案例中尤為突出，其口語表達完全由AAC輔助溝通系統替代，書寫由科技打字替代，複雜計算則由計算機替代。

重整 (Reorganization)

重新組織或排序學習目標的順序，以建立更穩固的基礎概念，有時會跨越不同年級的學習重點。阿祥的數學課程是此原則的最佳體現。教學團隊並未僵化地遵循年級順序，而是將加、減、乘、除四則運算的概念，利用1至10的小數字和計算機進行「重整」，使其能同步學習，從而建立對運算關係的整體理解，而非零碎、線性的程序記憶。

學習歷程的調整

此面向聚焦於「如何教」的層面，透過改變教學方法與互動模式，為學生搭建通往學習目標的鷹架。

  • 教學鷹架 (Instructional Scaffolding)： 這是最核心的策略之一，包含小步驟教學、提供具體操作的教具（如小芸和葉生的數學課）、運用圖形組織工具（如心智圖、樹狀圖），以及在過程中給予提示與回饋 。
  • 同儕中介教學 (Peer-Mediated Instruction)： 系統性地安排同儕擔任學習夥伴或小老師。例如，為小芸安排學伴，不僅能在課業上提供即時協助，更能透過互動增加其語彙量與學習自信 。
  • 多感官與體驗式學習： 強調動手做的學習方式，例如在數學課中使用積木等操作物，或是在語文課中融入遊戲、戲劇扮演等元素，以維持學生的學習動機與參與度 。「夢幻全壘打」的教學示例中，大量運用了角色扮演與視覺提示，即是此策略的體現 。
  • 學習策略教導： 明確地教導學生「如何學習」的後設認知策略。例如，教導小芸閱讀理解的策略（如找主題句、做摘要）以及數學應用題的解題步驟，旨在培養其獨立學習的能力 。
學習環境的調整

此面向關注「在哪裡學」的物理與社會心理情境，旨在移除環境中的障礙，創造一個安穩、支持性的學習氛圍。

  • 物理環境： 包含座位安排的巧思（如將需要提示的小芸或葉生安排在教師方便巡視的位置）、降低感官刺激（如簡潔的教室佈置以避免視覺干擾），以及建立結構化、可預測的空間佈局 。對於有肢體障礙的阿祥，則需確保教室動線的無障礙 。
  • 社會環境： 營造正向、友善的班級氛圍至關重要。具體作法包括安排友善的同儕擔任「小天使」、刻意創造讓學生展現優勢能力的成功機會以建立自信，以及給予具體、明確的讚美與鼓勵 。
  • 時間環境： 涉及時間管理與活動轉換的策略。例如，將一節課的內容分割成數個動靜交替的短活動，以維持注意力不足學生的專注；或在活動轉換前給予預告，幫助對改變較為敏感的學生（如自閉症的葉生）平穩過渡。
學習評量的調整

此面向處理「如何評量」的問題，強調評量方式應多元化與彈性化，以真實反映學生的學習成效，而非僅僅測量其應試能力。

  • 多元評量 (Diverse Assessment Methods)： 這是所有個案的核心建議。評量方式應超越傳統的紙筆測驗，納入口語測驗、實作評量、檔案評量（收集學生一段時間的作品以呈現進步軌跡）與課堂觀察等多種形式 。
  • 形成性評量 (Formative Assessment)： 增加評量的頻率，在教學過程中進行小單元的檢核，有助於教師即時了解學生的學習狀況，並提供及時的回饋與補救，避免學習困難的累積 。
  • 評量調整 (Assessment Accommodations)： 在評量過程中提供必要的支持。例如，為書寫或閱讀速度較慢的葉生提供延長作答時間；為無口語的阿祥允許使用AAC輔助溝通系統作答；或簡化試題的文字敘述，確保評量的是學科能力，而非學生的障礙本身 。
